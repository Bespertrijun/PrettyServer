# PrettyServer

<div align="center">
    
功能 | 描述
--- | ---
roletask | 替换演员为中文名
sorttask | 改变标题排序
synctask | emby、plex观看同步
…… | ……

</div>

### 1. 下载项目
```
git clone https://github.com/Bespertrijun/PrettyServer.git
cd PrettyServer
```

### 2. 修改配置文件

```
在 config.yaml 文件按需修改
```

### 3. 安装依赖
```
python -m pip install -r requirements.txt
```

### 4. 运行

```
#可以开个screen后台运行
python ./PrettyServer/main.py
```
